Object visibility

Calculate target altitudes for any night and observing location

Use this tool to calculate the altitudes of celestial objects for any night and observing location. Simply enter the date, time, and location details to see when your favorite objects will be visible in the night sky.

Visit my github page for more information by clicking the button below.

Usage

  • Visit my github page and a copy of the repository by cloning it.
  • Open the 'input.py' file and enter the location, observing date, target list, and optionally the telescope name.
  • Run the 'object_visibility.py' script to generate the altitude plot (this will also be saved as a pdf into the main directory).
